# Video-Conferencing-Tool-Python-ZeroMQ

Developed a real-time video, audio, and text conferencing tool supporting multiple users in one-on-one and group calls.

Implemented a centralized message broker using ZeroMQ PUB/SUB for asynchronous distribution of audio, video, and text streams between participants.

Managed backend logic for communication, synchronization, and channel separation, ensuring reliable message delivery and scalability for desktop applications.
